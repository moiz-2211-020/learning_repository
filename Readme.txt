git using steps
first clone your repo into the empty directory/folder
Than move to the repo folder using cd your-repo-folder-name
use command git add . or you can add particular file or folder by using its name like git add hello.html
Now after you add the file or folder in git repo its time to commit your changes in repo by using command git commit -m "write here what changes you made in your current code"
Now you need to check the status of the git repo with the command git status
And in the last step you will need to push your changes in repo using command git push
